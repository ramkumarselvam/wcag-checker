## Detailed Description

Run automated WCAG 2.1 AA accessibility audits on any web page with a single click.

WCAG 2.1 Checker uses axe-core, the industry-standard accessibility testing engine trusted by Microsoft, Google, and thousands of developers worldwide. Results appear instantly in a clean side panel — no context switching, no extra tabs.

✦ KEY FEATURES

• One-click audits — Run a full WCAG 2.1 AA check on any page instantly
• Side panel results — Issues grouped by impact: critical, serious, moderate, minor
• Element highlighting — Click "Highlight" to visually locate each issue on the page
• Auto-audit mode — Automatically re-run audits on every page navigation
• Summary stats — See pass/fail counts and an overall accessibility score
• Export JSON — Download full audit results for reporting and documentation

✦ WHO IT'S FOR

• Web developers checking their work
• QA testers running accessibility audits
• Designers validating accessible design choices
• Accessibility consultants auditing client sites
• Anyone building a more inclusive web

✦ PRIVACY FIRST

All processing happens locally in your browser. No data is collected, stored, or transmitted. No analytics, no tracking, no accounts. Your browsing stays private.

✦ POWERED BY AXE-CORE

Built on axe-core 4.9, the same engine behind Google Lighthouse accessibility audits. Reliable, accurate, and continuously updated.
